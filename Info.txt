255 bytes of memory 
4 general purpose registers 1 byte each , A , B , C , D , 
0 flag 

language

sub -  subtraction  (register name to subrtact to), (constant or register or Byte 01000011 )
store - storing (memory adress  name to store  to), (constant or register or Byte 01000011 )
load - (load to register from something )
jnz - ( jump if not zero )
jz - ( jump if zero )
print (parameter )
halt (stops program )
